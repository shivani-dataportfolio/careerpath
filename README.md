# BERT Career Assistant 🚀

An AI-powered platform designed to provide intelligent career guidance, automated resume analysis, and visual skill gap analysis. Whether you're a student, a career switcher, or a professional aiming for the next level, the BERT Career Assistant helps you chart your path with data-driven insights.

## ✨ Key Features
- **PDF Resume Parsing**: Extract skills, education, and experience automatically using our NLP engine.
- **Smart Recommendations**: Get matched with career paths that best suit your current profile.
- **Visual Skill Gaps**: Interactive charts showing exactly what skills you need to acquire.
- **Technical & Non-Technical Roles**: Comprehensive support for a wide range of career fields.
- **Secure Authentication**: Protected dashboard and profile management.

## 🛠️ Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Recharts, Lucide Icons.
- **Backend**: Node.js, Express, JWT, Multer, `pdf-parse`.

## 🚀 Quick Start (Local Development)

### 1. Install Dependencies
```bash
npm install          # Root (Frontend)
cd backend && npm i  # Backend
```

### 2. Run the App
- **Backend**: `cd backend && node server.js` (Port 8000)
- **Frontend**: `npm run dev` (Port 5173/Next available)
